# To-Do List

A Python-based To-Do List application for managing daily tasks and priorities.

## Features

* Add Tasks
* Remove Tasks
* Display Tasks
* Mark Tasks as Completed
* Priority Management
* Menu-Driven Interface
* Exception Handling

## Concepts Used

* Functions
* Lists
* Loops
* Match-Case Statements
* User Input Handling
* Exception Handling

## Language Used

* Python
